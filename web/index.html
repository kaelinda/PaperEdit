格式

小编辑器，也支持多种文件。

Markdown 支持源码与预览。结构化格式会在编辑区获得识别、语法着色和轻量校验。

  • Markdown、JSON、YAML、TOML、XML、plist
  • 纯文本与 shell 友好的编辑体验
  • 浅色、深色主题与原生 macOS 窗口质感

原生细节

看起来就该属于 Mac。

界面保持克制：统一标题栏、熟悉的侧边栏、清晰的编辑器字体，以及刚好够用的日常编辑控件。

  • 在合适的位置使用 SwiftUI 与 AppKit
  • 命令面板和设置窗口
  • 命令行打开：paper file-or-folder
